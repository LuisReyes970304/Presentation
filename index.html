What is a class?

We can understand a class like a blueprint or template for creating an object.

At the same time and Object is a virtual representation of a thing in the real world, with data and behavior.

Therefore, a class has:

  • Attributes

    • Attributes are the characteristics or properties of an object. They represent the data associated with the object and can be of various types, such as numbers, strings, lists, or even other objects.

  • Methods

    • Methods are the functions defined within a class that describe the behaviors or actions that an object of that class can perform

Example

escalera.py

Escalera class

This class is a template of a escalator that goes downstairs. It has five floors and as attributes and a piso_actual that at the beggining is empty because is waiting for the user to specify it.
This means that once the user select the floor this empty attribute is going to be replaced for that selection.

mostrar_piso

This is the method that show the current floor in which the user is, as you can see it includes the self.piso_actual so when we call this method is going to shows the current floor.

especificar_piso

This method is asking you for selecting the floor in which you are, so once you do it, the attribute self.piso_actual is going to be replaced for that option and then it will becomes in a integer.

bajar_seguir_bajando

This method wants to know if you want to keep going downstairs or if you want to stop

main.py

Here in the main.py is the place in which the Escalera class runs and get executed
This is where the "blueprint" becomes a reality. In this file, you instantiate the class (create the object) and start the execution flow.
in addition the main uses the time

time library

A library is a collection of pre-written code that you can "borrow" to perform specific.
In this case time is a library related to the concept of TIME and is used to give nice printing style to the code.

while loop

A while loop is a control flow statement that allows code to be executed repeatedly based on a given boolean condition.

input

The input function is used to receive data from the user. It allows the program to interact with the user and get information from them.
